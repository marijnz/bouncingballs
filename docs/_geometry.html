Geometry Import

NOTE: Even though this guide mainly uses .DAE files, it is highly recommended to use the .FBX format instead of Collada (.DAE). The ModelConverterD.exe can handle this format much better.

Importing Model Data to the GEP Engine

You will need

  • The model converter from the GEP/GPP Tools folder
  • A 3D modelling tool of your choosing that is able to export Collada (.DAE) or .FBX files, e.g.:
    • 3ds Max 2013 (e.g. educational plan available for HdM students)
    • Blender 3D Note: For animated or physical collision geometry you will still need 3ds Max or Maya!

Acquire a model or create your own using 3ds Max 2013. Important: Make sure that the textures are in the .dds format (DXT5)! If necessary, convert them.

Export the selected model from 3ds Max 2013:

Important: Select Z as the Up-Axis!

Use the ModelConverterD.exe to convert the .dae file (Collada) to the .thModel format. This can be done either on the command line or by using drag & drop.

Additional Information:

  • Copy the exported .thModel to %GEP%\data\models\
  • The texture paths in the model file need to be relative to the working directory of the game!
    • E.g. The textures that the model data/models/barbarian/barbarianStatic.thModel uses are referenced in its .FBX file like this:
      • data/models/barbarian/Barbarian_Belt_Low_d.dds
      • data/models/barbarian/Barbarian_Body_Low_d.dds

Creating a RenderComponent:

-- go => the game object instance created using GameObjectManager:createGameObject(...)
local renderComponent = go:createRenderComponent()
renderComponent:setPath("data/models/box.thModel")

Troubleshooting

  • All textures need to be quadratic and have to have a dimension of (2N)x(2N), e.g. 128x128, 256x256, etc.
  • All textures need to be in .dds format using the DXT5 compression setting.
  • When converting your model to a .thModel using ModelConverterD.exe, make sure your model has UV coordinates, i.e. your model is unwrapped.
    • Note: The model does not necessarily have to have a texture.

You will need

  • 3ds Max 2013 (e.g. educational plan available for HdM students)
  • Havok Content Tools available in the GEP/GPP Tools folder or at http://www.havok.com/try-havok
