Detailed Description

template<typename T>
struct gep::mat4_t< T >

a 4x4 matrix

uses a column-major data layout [ 0] [ 4] [ 8] [12] [ 1] [ 5] [ 9] [13] [ 2] [ 6] [10] [14] [ 3] [ 7] [11] [15]

definition is: |m00 m01 m02 m03| |m10 m11 m12 m13| M = |m20 m21 m22 m23| |m30 m31 m32 m33|
