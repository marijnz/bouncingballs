Rigid Body Import and Usage
  1. Download and install 3ds Max 2013
  2. Download and install Havok Content Tools for 3ds Max 2013 from the GEP/GPP Tools folder or from http://www.havok.com/try-havok
  3. Create model in 3ds Max with 3ds Max's coordinate system
  4. Select the object you want to export
  5. Click Havok Content Tools -> Physics -> Create Rigid Body in the tools section
    • Important: This has to be done for every single object you want to have physics
  6. Click Havok Content Tools -> Export in the tools section
  7. Clear the configuration set
  8. Open the Physics section on the left and add [Physics 2012] Create Rigid Bodies to the configuration set.
  9. Open the Core section on the left and add Write to Platform to the configuration set
    • Important: The Transform scene configuration seems to not work, don't try to use this
    • Important: Don't change the order of the two configurations
  10. Select the Write to Platform configuration on the right and select Format -> Binary and Format -> Packfile in its properties section. Use the filename property to configure the export path.
  11. Hit Run Configuration next to the configuration set and watch the export happen.
  12. Use the exporter file (with the .hkx file extension) in your game in the following manner:
my_object = GameObjectManager:createGameObject("my_object")
my_object.pc = scene.transformation_test:createPhysicsComponent()
local cinfo = RigidBodyCInfo()
cinfo.shape = PhysicsFactory:loadCollisionMesh("data/collision/my_object.hkx")
cinfo.motionType = MotionType.Fixed
my_object.pc.rb = my_object.pc:createRigidBody(cinfo)
