1 Rigid Body Import and Usage {#RigidBody}
2 ===========================
3 
4 1. Download and install **3ds Max 2013**
5 2. Download and install **Havok Content Tools** for 3ds Max 2013 from the [GEP/GPP Tools folder][GPP_Tools] or from http://www.havok.com/try-havok
6 3. Create model in 3ds Max with 3ds Max's coordinate system
7 4. Select the object you want to export
8 5. Click **Havok Content Tools -> Physics -> Create Rigid Body** in the tools section
9  - <strong><span style="color:#ff0000">Important:</span></strong> *This has to be done for every single object you want to have physics*
10 6. Click **Havok Content Tools -> Export** in the tools section
11 7. Clear the configuration set
12 8. Open the **Physics** section on the left and add <strong>[Physics 2012] Create Rigid Bodies</strong> to the configuration set.
13 9. Open the Core section on the left and add **Write to Platform** to the configuration set
14  - <strong><span style="color:#ff0000">Important:</span></strong> *The Transform scene configuration seems to not work, don't try to use this*
15  - <strong><span style="color:#ff0000">Important:</span></strong> *Don't change the order of the two configurations*
16 10. Select the **Write to Platform** configuration on the right and select **Format -> Binary** and **Format -> Packfile** in its properties section. Use the filename property to configure the export path.
17 11. Hit **Run Configuration** next to the configuration set and watch the export happen.
18 12. Use the exporter file (<em>with the .hkx file extension</em>) in your game in the following manner:
19 
20 ~~~{.lua}
21 my_object = GameObjectManager:createGameObject("my_object")
22 my_object.pc = scene.transformation_test:createPhysicsComponent()
23 local cinfo = RigidBodyCInfo()
24 cinfo.shape = PhysicsFactory:loadCollisionMesh("data/collision/my_object.hkx")
25 cinfo.motionType = MotionType.Fixed
26 my_object.pc.rb = my_object.pc:createRigidBody(cinfo)
27 ~~~
